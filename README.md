# 🌿 ISPANI UI

### Earn. Save. Grow Together.

**South African Digital Labour Marketplace — Frontend UI**

## Live Demo
**https://ispani-sa.codewords.run**

## Pages
- `/` — Login (phone +27 + password)
- `/register` — Registration with password strength meter

## Tech Stack
- Next.js 16 + TypeScript
- Tailwind CSS 4 + shadcn/ui
- Framer Motion animations

## Design System
- Primary: Emerald Green (#10B981)
- Fonts: Plus Jakarta Sans + DM Sans
- Dark green gradient login theme
- Responsive across all devices
